# Security Policy

## Supported Versions

This project is in active early development. Security fixes are applied to the latest code on the default branch.

## Reporting a Vulnerability

If you discover a security issue, please do not open a public issue.

- Email: security@polis-project.org
- Subject: `[SECURITY] polis-mobile vulnerability report`

Please include:
- A clear description of the issue
- Reproduction steps
- Potential impact
- Suggested mitigation (if known)

We will acknowledge reports as soon as possible and work on a fix.
